### Development Installation
```bash
yarn set version berry
yarn
```

Then, to work on all packages, setup these commands to all run simultaneously:

```bash
# in root
npx nodemon -x "npx @data-story/core server"

# in core
yarn tsc --watch
npx webpack --watch

# in ui
yarn tsc --watch
yarn watch:css
npx webpack --watch

# in docs
yarn dev
```
